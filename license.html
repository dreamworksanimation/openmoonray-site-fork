Contributor License Agreement

Your suggestions and contributions are welcome! Bug reports and suggestions can be added as "Issues" on the MoonRay GitHub repository. Developers who wish to contribute code to be considered for inclusion in the MoonRay distribution must first complete this Contributor License Agreement and submit it to moonray@dreamworks.com

We prefer code submissions in the form of pull requests to the GitHub repository. All code should adhere to the moonray coding standards, and every commit must be signed off. That is, every commit log message must include a "Signed-off-by" line (generated, for example, with "git commit --signoff"), indicating that the committer wrote the code and has the right to release it under the AL-2.0 license. See http://developercertificate.org/ for more information on this requirement.
